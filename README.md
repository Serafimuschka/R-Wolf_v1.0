# R-Wolf
Простой и нетребовательный графический движок, написанный на C++ и Windows API.
***
## Системные требования
* ОС не старше Windows 7 SP1, x86 или x64
* Видеокарта с поддержкой DirectX 10
***
## Требования для разработки
* Visual Studio IDE 2019 (v142)
* Windows 10 SDK v. 10.0.17763.0
* DirectX SDK June 2010
***
## Обратите внимание

<https://github.com/Serafimuschka/triangles_game>
